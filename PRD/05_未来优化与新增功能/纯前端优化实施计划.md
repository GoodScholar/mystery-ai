## 二、各功能技术方案

### F-05 超长对话虚拟列表

- 轻量级 VirtualList 类，动态行高，仅渲染可视区域 ± 缓冲区
- 替换 `#chat-messages` 容器为虚拟列表管理
- 切换 NPC 时销毁/重建实例

### F-09 PWA

- 新增 `manifest.json`（standalone模式）和 `sw.js`（Cache-First 策略）
- index.html 添加 manifest 引用和 theme-color meta
- main.js 注册 Service Worker

### F-11 草稿箱

- `draft-manager.js` 管理 localStorage 多草稿
- Step 1/2 自动保存（debounce 500ms）
- 进入工坊时检测未完成草稿并提示恢复

### F-01 深色/浅色模式

- `[data-theme="light"]` CSS 变量覆盖
- ThemeManager 持久化主题偏好
- 设置弹窗增加 Dark/Light/Auto 切换

### F-06 存档导入导出

- state.js 新增 exportSave/importSave 方法
- 首页新增导入按钮和导出操作

### F-07 线索背包增强

- 按来源 NPC 筛选、搜索、标记重要
- 关键词聚合关系列表

### F-10 实时预览

- Step 2 分屏预览面板
- 设定变更实时更新

### F-04 环境音效

- Web Audio API 合成简单氛围音效
- 音量控制持久化

### F-02 转场动效

- 多种动画类型：fade/slide/zoom

### F-03 打字机音效

- 流式输出时 Web Audio 合成短脉冲声

### F-08 字号调节

- CSS 变量 `--base-font-size` 控制
- 设置中增加字号滑块
